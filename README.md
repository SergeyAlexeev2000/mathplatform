# Highest Math Platform (FastAPI + Frontend)

A tiny, extendable “Moodle-like” platform for higher math.  
Backend: FastAPI (HTTP API, Swagger docs)  
Frontend: static HTML/CSS/JS (served separately)

### Prereqs
- Python 3.9+ (3.10/3.11 fine)
- Git
- (Optional) VS Code
